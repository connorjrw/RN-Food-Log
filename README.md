# React Native Food Log

Simple meal tracking app created with React Native

## Setup / Installation
- install and start mongodb locally(https://docs.mongodb.com/guides/server/install/)
- run `npm install`
- run `cd ios && pod install`
- edit config.js to match local environment

## Run App
- `react-native run-ios`
- `node server.js`
- `http-server ./images`
